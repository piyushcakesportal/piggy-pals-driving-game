# Pip & Pals: Road Trip

A lightweight, dependency-free Android driving game for playtesting. Choose an
animal driver, drag left or right to steer, collect stars and apples, and avoid
road hazards. The game becomes faster over time and stores the best score on the
device.

## Controls

- Choose Pip, Benny, or Ellie on the start screen.
- Tap **Drive**.
- Drag anywhere on the road to steer.
- Use the top-right pause button to pause or return home.

## Build an APK

Push the `agent/piggy-road-trip` branch or run the GitHub Actions workflow manually.
Download the `pip-and-pals-road-trip-apk` artifact and install `app-debug.apk`
on an Android 6.0 or newer device.
